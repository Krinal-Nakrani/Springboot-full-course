# Springboot-full-course
Spring Boot Full Course is a hands-on, end-to-end Advanced Java training repository focused on Java EE, Spring Framework, and Spring Boot. This repository is designed for beginners to intermediate Java developers who want to build real-world, enterprise-ready applications using Spring Boot, REST APIs, and databases.

**Prerequisites:**
  Core Java (OOP concepts)
  Basic SQL knowledge
  Understanding of HTTP & Web concepts (recommended)
  Willingness to code and explore 😊

**Tools & Technologies Used**
🔹 IDEs
IntelliJ IDEA (Community / Ultimate)
Spring Tool Suite (STS – optional)

🔹 JDK 
Java JDK 11 or later

🔹 Spring Ecosystem
Spring Initializr

🔹 Database
MySQL

🔹 Testing
Postman
